Validación de los resultados

En este último paso se pueden validar los resultados del algoritmo de clasificación brindando las clases correctas para cada seguidor.

Las clases manualmente elegidas para cada uno de tus seguidores deben ser ingresadas en un archivo .csv que respete el formato de este template (clickea aquí para descargarlo).

La Matriz de Confusión aporta métricas para evaluar al clasificador considerando las predicciones positivas y negativas, y los valores reales positivos y negativos para cada clase.
