Escaneo de Followers

El primer paso consiste en el recorrido de la lista de tus seguidores y la descarga de sus imágenes de perfil y posts.

Este proceso está automatizado con Selenium Webdriver. Selenium es un entorno de pruebas de software para aplicaciones basadas en la web que emula acciones de usuarios.

Clasificación por imágenes

El Clasificador consiste en un modelo previamente entrenado con una determinada cantidad de categorías de clasificación y su algoritmo clasificador.

Cada imagen persistida en el paso anterior será analizada por el clasificador y categorizada dentro de las clases que contenga el modelo.

El clasificador está implementado como una red neuronal convolucional (CNN).

Validación de los resultados

En este último paso se pueden validar los resultados del algoritmo de clasificación brindando las clases correctas para cada seguidor.

Las clases manualmente elegidas para cada uno de tus seguidores deben ser ingresadas en un archivo .csv que respete el formato de este template (clickea aquí para descargarlo).

La Matriz de Confusión aporta métricas para evaluar al clasificador considerando las predicciones positivas y negativas, y los valores reales positivos y negativos para cada clase.
